## 免费项目实战课程

1. 不知道简历的项目要写什么？想边做项目边学习新技能？本项目致力于给程序员提供网络上最好的**免费项目实战课程**。
2. 为了达到这个目标，我们搜索了国内外大量的不同平台的网络课程以及技术博客，并经过初步的筛选，免费的项目并不代表质量低，有时候免费的反而才是最好的。
3. 网络课程质量参差不齐，学习前可以先对比再谨慎选择，若课程有涉及以下几种情况的话，请提交 issue。
    - 涉及大量广告
    - 需要付费才能学习完整课程
    - 过时，不适合当前开发环境
    - 错漏百出，质量非常低
3. 课程排序按照字符串从小到大升序排列
